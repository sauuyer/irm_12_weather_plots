# Met Intercomparison Graphs for Irminger 12

Ship data from Irminger 12 comes from the R/V Revelle (where usually data comes from the R/V Armstrong for most CGSN Irminger Cruises). 
Ship data from the Revelle is SAMOS. Variable differences are accounted for wind-related values/calculations. See notebook markdown for
more details/

These graphs were created for use in the Irminger 12 Cruise Report in collaboration with the Chief Sci and Documentation lead for said 
cruise.
